The IIFE Is:

An IIFE (Immediately Invoked Function Expression) is a JavaScript function that runs as soon as it is defined.

It is a design pattern which is also known as a Self-Executing Anonymous Function

we use IIFE to:

- Avoid polluting the global namespace

- using IIFE in this case is better than using a function declaration or a function expression.

- We would also use IIFE to create private and public variables and methods.

example of IIFE:

(function(text) { document.getElementById("result").innerHTML = text; })("hi i IIFE Example");
